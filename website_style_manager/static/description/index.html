Apply site wide styles to various HTML tags

Site wide styles

Apply site wide styles to various HTML tags.

Instructions
1. Go to Website Admin->Configuration
2. Open the website form view
3. Select HTML tags and apply styles
4. These styles will be automatically applied to any page using the standard website layout(website.layout)
5. You can turn off custom styles via the customize menu in the website editor
