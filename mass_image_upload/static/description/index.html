Description

Designed to migrate images from proprietary system which store images in the filesystem

Instructions

1. Go to /mass/image/upload as an administrator
2. Pick the field that matches the filename of your images
3. Upload the images
4. View the error log to see if any images didn't upload
5. You can view real time progress updates via the javascript console
