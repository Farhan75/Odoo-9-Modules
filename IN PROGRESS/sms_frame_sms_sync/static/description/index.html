Share your Odoo instance with others

*NOTE* It is advised to have your template database setup with a restricted SAAS user who can't access the app store since a failed install / uninstall can crash all database / the instance

I also advise editting 'controllers/main.py' to customise the database creation process

You can change your template database by going to /openerp/tools/config.py and changing "--db-template"
