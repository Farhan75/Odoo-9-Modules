Description

Save pages as templates and reuse them later

You can save a page as a template by going to content->Save as Template

You can create a new page from an existing template by going to content->New page templates

You can add preview images to your templates in website Admin->Website Templates

Future Features

1. Auto generate preview image templates
2. Revamp template preview window so you can see a big picture of what a template looks like
